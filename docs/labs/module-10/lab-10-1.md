# Lab 10.1: Enabling HTTP/3 (QUIC)

**Goal:** Test the next gen protocol.

*Requires a modern Nginx or Caddy.*

**Caddy is easiest:**

1.  Run Caddy:
    `caddy file-server --domain localhost --listen :443`
2.  Caddy enables HTTP/3 by default.
3.  Test with a browser (Chrome shows "h3" in protocol column).
4.  Test with curl:
    `curl --http3 -v https://localhost`

**Observation:**
You will see UDP traffic on port 443, not TCP.
